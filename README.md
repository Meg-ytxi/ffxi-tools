# DD装備管理表

FFXI（ファイナルファンタジーXI）のDynamis-Divergence（DD）装備の収集進捗を管理するためのWebアプリです。


## 概要

全22ジョブ × 5部位（Head / Body / Hands / Legs / Feet）× 2段階（+2 / +3）、合計220個の装備チェックボックスを一覧表示します。チェック状態はブラウザのLocalStorageに自動保存されます。

### 主な機能

- **進捗バー** — 全体の収集率をリアルタイムで表示
- **ジョブ別進捗** — 各ジョブの達成数・達成率を行ごとに表示
- **列ハイライト** — 全ジョブ分が揃った部位列を自動でハイライト
- **完了行ハイライト** — すべての装備が揃ったジョブ行を強調表示
- **エクスポート / インポート** — 進捗データをJSONファイルで保存・復元
- **統計表示** — 完了・進行中・未着手のジョブ数と部位別集計をポップアップ表示
- **LocalStorageテスト** — ブラウザの保存機能が正常か確認
- **リセット** — 全進捗を一括クリア


## 動作環境

| 項目 | 内容 |
|------|------|
| 実装 | 単一HTMLファイル（外部ライブラリなし） |
| 依存フォント | Google Fonts（Rajdhani / Noto Sans JP） |
| データ保存 | ブラウザのLocalStorage |
| 推奨ブラウザ | Chrome / Firefox / Safari / Edge（最新版） |

> LocalStorageはブラウザ・デバイスごとに独立しています。別のブラウザや端末でデータを引き継ぐにはエクスポート／インポート機能をご利用ください。


## 使い方

1. GitHub Pagesで公開されたURLをブラウザで開く
2. 収集済みの装備のチェックボックスをオンにする（自動保存されます）
3. データを別デバイスに移したい場合は「Export JSON」でファイルを保存し、移行先で「Import JSON」から読み込む

# FFXI Tools

ファイナルファンタジーXI（FFXI）向けのブラウザツール集です。  
GitHub Pages で公開しており、インストール不要・ブラウザだけで動作します。

---

## ツール一覧

### [DD Gear Tracker](./dd-gear-tracker.html)
デュナミス～ダイバージェンス～（DD）装備の収集進捗を管理する表です。  
全22ジョブ × 5部位 × +2/+3 の計220個をチェックボックスで管理できます。

**主な機能**
- 全体・ジョブ別の進捗バーをリアルタイム表示
- 全ジョブ分が揃った列・行を自動ハイライト
- 進捗データのエクスポート / インポート（JSON）
- データはブラウザの LocalStorage に自動保存

---

### [Synthesis Cost Calculator](./synthesis-cost-calculator.html)
FFXI の合成原価を計算するツールです。  
素材の購入費・成功回数・残素材の価値をもとに、成功品1個あたりの原価や損益分岐点を算出します。

**主な機能**
- 素材費・成功回数・残素材から原価を自動計算
- 損益分岐点・推奨売値（利益10%）の表示
- 売値を入力して利益をシミュレーション
- 計算結果を最大10件まで LocalStorage に保存

---

## 動作環境

- **実装：** 単一 HTML ファイル（外部ライブラリなし）
- **データ保存：** ブラウザの LocalStorage（サーバーへの送信なし）
- **推奨ブラウザ：** Chrome / Firefox / Safari / Edge（最新版）

> LocalStorage はブラウザ・デバイスごとに独立しています。  
> 別の端末にデータを移す場合はエクスポート／インポート機能をご利用ください。
